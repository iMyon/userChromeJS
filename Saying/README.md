saying
============

 - 地址栏显示自定义语句，根据网址切换。
 - 目前可自动获取并更新的有VeryCD标题上的，和hitokoto API。
 - 每次关闭浏览器后数据库添加获取过的内容，并去重复。
 - 左键图标复制内容，中键重新获取，右键弹出菜单。
 - 其他详细设置请参照脚本内的设置。
 - 自定义内容可在saying.json自己添加，此文件脚本只读取，格式请参照例子。
 - 其他同hitokoto.uc.js脚本。
 - 数据库位置：

		Chrome\lib\saying.json
		Chrome\lib\hitokoto.json
		Chrome\lib\veryCD.json
